Integrates Elementor forms with Brevo API. Dynamic field mapping for all contact attributes. Multilingual support included.

== Description ==

ML Brevo for Elementor Pro is a powerful plugin that seamlessly integrates your Elementor forms with the Brevo (formerly SendinBlue) email marketing platform.

**Key Features:**

* **Dynamic Field Mapping**: Map any Elementor form field to any Brevo contact attribute
* **Global API Configuration**: Set your API key once and use it across all forms
* **List Management**: Subscribe contacts to specific Brevo lists
* **Unsubscribe Action**: Allow contacts to unsubscribe from lists
* **Multilingual Support**: Available in Italian, French, German, and Spanish
* **Debug Logging**: Comprehensive logging for troubleshooting
* **Cache Management**: Efficient caching of Brevo attributes and lists

**Supported Brevo Features:**

* Contact creation and updates
* Custom contact attributes
* List subscription/unsubscription
* Double opt-in support
* Error handling and logging

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ml-brevo-for-elementor-pro` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings â†’ Brevo to configure your API key.
4. Add the Brevo integration action to your Elementor forms.

== Frequently Asked Questions ==

= Do I need a Brevo account? =

Yes, you need a Brevo account and API key to use this plugin.

= Is this compatible with Elementor Pro? =

Yes, this plugin is designed to work with Elementor Pro forms.

= Can I map custom fields? =

Yes, the plugin supports mapping to all your Brevo contact attributes, including custom ones.
